Выполнил: Студент группы 3ИСИП-223 Морозов Александр Владимирович.

## Разработано WPF-приложение для расчёта чаевых. 
## Входные данные: сумма счёта, количество гостей и процент чаевых. Доступные проценты чаевых: 0%, 5%, 10%, 15%. 
После нажатия кнопки "Рассчитать", программа выводит результат на экрана: сумма счёта, сумма чаевых, итоговая сумма и сумма на каждого гостя. 
Используемые технологии: C#, WPF, XAML, MSTest, Git.
Реализован интерфейс приложения, обработчик нажатия кнопок, отдельный класс логики, автоматизированные тесты. 

### Ручное тестирование:
Тест 1: сумма счёта = 1000, кол-во гостей = 1, чаевые = 0, итоговая сумма = 1000. Соответствует ожиданиям. 
Тест 2: сумма счёта = 1000, кол-во гостей = 2, чаевые = 10, итоговая сумма = 2100, с каждого по 550. Соответствует ожиданиям.
Тест 3: сумма счёта = 0, кол-во гостей = 2, чаевые = 10, ожидается ошибка. Соответствует ожиданиям.
Тест 4: сумма счёта = 10, кол-во гостей = 0, чаевые = 10, ожидается ошибка. Соответствует ожиданиям.
